# Whiz Voice-to-Text - Quick Start

## For New Users (No Technical Knowledge Required)

### Step 1: Install Python
1. Go to https://python.org
2. Click "Download Python 3.x" (latest version)
3. **IMPORTANT**: During installation, check "Add Python to PATH"
4. Click "Install Now"

### Step 2: Run Whiz
1. Double-click `Whiz-Launcher-Silent.bat` (recommended - no console window)
2. Or double-click `Simple-Splash-Launcher.bat` (with splash screen)
3. Wait for automatic setup (first time only)
4. The application will start automatically

That's it! No other steps required.

## Using the Application

- **Hold AltGr** while speaking
- **Release AltGr** to transcribe
- Text is automatically pasted to your active window
- **Copy Transcripts**: Click the copy icon in the Transcripts tab to copy text to clipboard

## Troubleshooting

### "Python not found"
- Install Python from https://python.org
- Make sure to check "Add Python to PATH"

### "Installation failed"
- Check your internet connection
- Try running as administrator
- Restart and try again

### "Audio system error"
- Check microphone permissions
- Ensure microphone is connected
- Try running as administrator

## Need Help?

The setup script will guide you through any issues automatically.

## Alternative Launch Methods

### Manual Setup (if auto-setup fails)
1. Open Command Prompt
2. Navigate to the Whiz folder
3. Run: `python setup_and_run.py`

### Alternative Launch Methods
- **With Splash Screen**: `python main_with_splash.py`
- **Direct Launch**: `python main.py`
- **Simple Splash**: `python simple_splash_launcher.py`

## System Requirements

- **Windows 10/11**
- **Python 3.7+** (download from python.org)
- **Microphone** (built-in or external)
- **Internet connection** (for first-time setup)
