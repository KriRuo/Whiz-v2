Whiz Voice-to-Text Application
=============================

Installation Instructions:
1. Run install-whiz.bat as Administrator
2. The application will be installed to your Desktop
3. Run Whiz.exe to start the application

Uninstallation:
1. Run uninstall-whiz.bat as Administrator

System Requirements:
- Windows 10/11
- Microphone for voice input
- 4GB RAM minimum

For support, visit: https://github.com/your-repo/whiz
